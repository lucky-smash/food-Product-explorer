A React-based food exploration app using the OpenFoodFacts API.
Users can search food products and view detailed nutritional information.

Features
- Search food products by name
- View product details with nutrition facts
- Responsive UI
- Client-side routing

Tech Stack
- React
- React Router
- Tailwind CSS
- OpenFoodFacts API

How to Run
1. Clone the repo
2. npm install
3. npm run dev
